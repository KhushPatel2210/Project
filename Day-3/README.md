# Day-3 Project

## Description

Basic setup of .NET Web API using Visual Studio 2022.  
Learned how to create REST API routes like GET, POST, PUT, PATCH, DELETE.  
We are using **Swagger** to test the API easily.

## Folder Structure

- `BooksApi/` : Main .NET API project
  - `Controllers/` : API route controllers
  - `Models/` : Data models
  - Other folders are auto-generated by Visual Studio

## How to Run

1. Open project in Visual Studio 2022.
2. Press F5 or click Start to run.
3. Open Swagger UI at `https://localhost:<port>/swagger` to test API routes.

## Basic REST Routes

- **GET** — Get data
- **POST** — Add data
- **PUT** — Update full data
- **PATCH** — Update partial data
- **DELETE** — Delete data
